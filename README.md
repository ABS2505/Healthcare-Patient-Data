# Healthcare-Patient-Data
Healthcare Patient Data Analysis

 Project Overview

This project focuses on analyzing healthcare patient data to identify common diseases, demographic patterns, and high-risk patient groups. The analysis helps in understanding how diseases are distributed across different age groups and genders.
 Objective
 
 Analyze patient data to identify common diseases
Study demographic risk factors (Age, Gender)
Detect high-risk patient groups
Visualize insights using charts

 Dataset

Dataset used from Kaggle:
 Healthcare Dataset 
https://www.kaggle.com/datasets/prasad22/healthcare-dataset ⁠
